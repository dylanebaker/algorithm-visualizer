# Sorting Algorithm Visualizer

An interactive sorting algorithm visualizer built with Pygame. Watch six algorithms sort a randomized list in real time, with color-coded bar highlights and pitch-mapped audio feedback.

## Algorithms
- Bubble Sort
- Insertion Sort
- Selection Sort
- Quick Sort
- Merge Sort
- Heap Sort

## Controls
| Key | Action |
|-----|--------|
| `Space` | Start sorting |
| `R` | Reset with a new random list |
| `A` / `D` | Ascending / Descending order |
| `B` | Bubble Sort |
| `I` | Insertion Sort |
| `S` | Selection Sort |
| `Q` | Quick Sort |
| `M` | Merge Sort |
| `H` | Heap Sort |
